# Examples Index

All examples are original Manim CE scenes and avoid `MathTex` so they can render without a host TeX install.

## `examples/scenes/basic_transform.py`

Scene: `BasicTransform`

Small smoke test: title, square, circle, and transform.

## `examples/scenes/linear_transform.py`

Scene: `LinearTransformDemo`

Grid plus basis vectors under a matrix transform. Good starting point for linear algebra visuals.

## `examples/scenes/function_story.py`

Scene: `FunctionStory`

Axes, plotted curve, moving dot, tangent marker, and value tracker.

## `examples/scenes/attention_flow.py`

Scene: `AttentionFlow`

Transformer-style token-to-token attention visualization with weighted edges.
